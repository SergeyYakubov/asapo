<TabItem value="cpp">

```cpp content="./versioned_examples/version-21.09.0/cpp/produce.cpp"
```

Compile e.g. using CMake and execute. You might need to point cmake (with CMAKE_PREFIX_PATH) to asapo installation and curl library if installed to non-standard location.

```shell content="./versioned_examples/version-21.09.0/cpp/CMakeLists.txt" snippetTag="#producer"
```

```shell
$ cmake . && make
$ ./asapo-produce
```

</TabItem>
